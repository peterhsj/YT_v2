YS-200 智慧音頻主機

一機二用

一個主機可以做為麥克風擴音。
也可以做為手機、平板、電腦的無線音頻接收器。

多機共享

經由主機內的配對記憶與智慧搜尋，
可以共用在不同麥克風或手機、筆電，操作簡單快速。
1 2 3 4 5

  1. 堅固 6.3mm 插頭 :

    可達10,000次插拔不損壞。傳統與特製 L 型插頭，可依設備環境選購。

  2. 特殊線扣設計 :

    經由電纜線前結插頭與主機，兩端的線扣更加強產品的一體性，增加使用時的彈性，也避免損壞。

  3. 插頭與主機連接電纜 :

    使產品更適應環境的需要，減少碰撞，易於使用。

  4. 電源鍵 / 功能鍵 / LED燈號顯示 :

    電源與功能鍵合而為一，以達輕薄短小目的、操作依舊。
    LED 清晰顯示方式，可輕易了解麥克風狀態。

  5. 一體化 無插座設計 :

    無外接線，避免接觸不良造成雜音。

產品特色

  • 智慧連結 :

    可配對/記憶多個設備，完全不需回工廠設定。

  • 自動連接 :

    免對頻。開機後智慧搜尋，自動對頻 不串音，自動連結立即使用。

  • 抗干擾 :

    先進跳頻技術(FHSS)避免如 Wifi其他無線設備干擾，一對一連結。

  • 一對一連結 :

    保證不串音不干擾。

  • 外觀 :

    人體工學設計，強調使用便利與外型美觀。

  • 獨家 L型 麥克風插頭 :

    特製 L型麥克風插頭可完全避免傳統麥克風插頭突出易被撞擊問題。

  • 重量 :

    重量 40g。

  • 距離 :

    15 公尺以上。

  • 使用時間 :

    20hr.；長效鋰電池與加日本電池保護晶片，延長電池壽命。
